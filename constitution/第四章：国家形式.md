# 国家形式的含义
## 国家形式的概念
国家统治阶级实现国家权力的形式  
包括国家政权组织形式、国家结构形式和国家象征
## 与国家性质的关系
国家性质决定国家形式，国家形式反映国家性质
## 与宪法的关系
宪法全面规定了国家形式，国家形式是宪法文本的重要内容
# 国家政权组织形式
## 政权组织形式含义
特定社会的统治阶级采取何种原则和方式去组织反对敌人、保护自己、治理社会的政权机关  
包括国家机关的组成、职权以及相互关系等内容
## 政权组织形式的种类及特征
根据国家权力归属于世袭的个人还是任期限制的多数人为标准分为：君主政体和共和政体
- 君主政体  
    存在君主，君主是世袭的，国家最高权力归属于君主  
    根据君主是否享有实际权利分为：二元君主立宪政体和议会君主立宪政体
    - 二元君主政体  
        国家有宪法和议会，但君主集立法、行政、司法和军事大权于一身，是权力中心的最高实际统治者  
        宪法是钦定的，是君主意志的反映  
        议会是君主的咨询机构，立法权是形式的，君主有否决议会立法的权力，还通过任命指定议员控制议会  
        内阁是君主行使行政权的机构，首相由君主任命  
        例如：1871-1918德意志帝国、1889-1945日本、1980s约旦沙特科威特摩洛哥等国家
    - 议会君主立宪制  
        议会是国家最高立法机关和权力机关，君主不直接支配国家政权第整体形式  
        内阁从议会产生，通常由议会中多数党或执政党联合组阁，对议会负责，君主只履行任命手续  
        如果内阁失去议会信任，必须辞职或者提请君主解散议会
- 共和政体  
    国家最高权力归属于有一定任期的公职人员，公职人员是由人民选举产生的  
    根据行政机关与立法机关关系以及行政机关内部组成和运行方式可以分为：总统制、议会内阁制、半总统制、委员会制、人民代表大会制
    - 人民代表大会制  
        指依据宪法和有关法律规定，由人民按照一定原则和程序，选举人民代表组成全国人大和地方各级人大，作为国家权力机关，再由各级权力机关产生同级其他国家机关，这些国家机关对人民负责，受人民监督的一种国家政权组织形式  
        人民代表大会制度描述了权力来源、权利分配、权力运行，是我国的宪政体制，根本民主政治制度  
        人大制度还需要进一步完善  
        - 逻辑起点：国家一切权力属于人民  
        - 前提：选民选举产生代表组成全国人大及地方人大  
        - 核心：以人民代表大会为基础用选举、立法两种方式建立全部国家机构
        - 关键：人民代表大会对人民负责，受人民监督
        - 人民代表大会制度为什么是根本政治制度  
            1. 直接全面反映我国阶级本质  
            2. 该制度产生于我国革命斗争中，是其他制度赖以建立的基础
            3. 该制度反映了我国政治生活的全貌
        - 人大制度的优越性  
            1. 便于人民管理国家  
            2. 便于统一行使国家权力，有效地进行国家管理
            3. 该制度能在保证中央的统一领导下发挥地方的主动性和积极性
            4. 保证了少数民族参与国家管理，有助于实现民族平等
# 国家结构形式
## 国家结构形式的含义
统治阶级依据一定的原则和方式划分国家的内部区域，调整国家整体与组成部分、中央与地方以及地方与地方之间关系的制度  
体现国家权力的纵向配制原则  
核心是对国家权力纵向配置的原则和方式  
宪法与国家形式结构的关系：国家结构形式决定宪法内容，国家结构形式影响宪法纵向配置国家权力，建立相应国家机关；宪法文本规定国家结构形式
## 国家结构形式的种类及特征
- 单一制  
    - 概念  
        指由若干不具有独立性的地方组成的同一主权国家
    - 特征  
        1. 法律制度上看，只有一部宪法，所有机关设置以宪法为依据  
        2. 政权组织形式上看，只有一套政府体质
        3. 权力配置上看，所有权力归属于中央，地方权力来源于中央
        4. 对外关系上看，中央政府代表国家公民有统一国籍
    目前属于单一制的国家有：中英法日意西葡
- 复合制  
    - 概念  
        由两个或以上国家基于共同目的而联合组成的国家联盟，可分为联邦和邦联
    - 联邦制  
        是由两个或以上成员国组成的国家  
        - 特征  
            1. 从国家法律体系看，除有联邦的宪法外，各成员国还有各自的宪法
            2. 从国家机构看，除设有联邦立法机关、政府和司法系统外，各成员国还有各自的立法机关、政府和司法系统
            3. 从联邦与各成员国职权划分看，其职权划分由联邦宪法作出具体规定，剩余权利一般归成员国保留
            4. 从对外关系看，有些国家允许成员国有一定外交权，联邦公民既有联邦国籍又有成员国国籍
    - 邦联制  
        是由几个独立的国家为了一定的目的而结成的比较松散的国家联合  
        - 特征  
            1. 不是完全意义上的国家，不是国际法主体
            2. 各成员国保留自己的主权，经过平等协商，把各自的一部分权力委托给联邦机构
            3. 联邦机构没有统一的宪法，没有最高立法机关和行政机关，也没有统一军队、赋税、预算、国籍等
            4. 联邦议会是协商机关，决议必须经过成员国认可才有约束力
            5. 各成员国可以自由退出联邦
    - 独立国家联合体  
        由前苏联大多数共和国组成的进行多边合作的独立国家联合体  
        除波罗的海三国外，其他12个前苏联加盟共和国阿塞拜疆、亚美尼亚、白俄罗斯、格鲁吉亚、吉尔吉斯斯坦、摩尔多瓦、哈萨克斯坦、俄罗斯、乌兹别克斯坦、乌克兰、塔吉克斯坦和土库曼斯坦均为独联体正式成员国，2005土库曼斯坦退出，2009格鲁吉亚退出
        - 机构  
            设国家元首理事会，政府首脑理事会，跨国议会大会，协调协商委员会等  
            总部在白俄罗斯首都明斯克，工作语言为俄语
## 我国国家形式结构原因及特征
- 原因  
    主要是历史和民族原因，秦汉以来实行中央集权绝大多数是汉族，8%是少数民族，民族之间大杂居、小聚居
- 我国单一制特征  
    1. 包含34个省级行政区，23个省，5个自治区，4个直辖市、2个特别行政区
    2. 包含若干层次：两个特别行政区，三级制和四级制
    3. 包含多种中央与地方权力模式：特别行政区、民族自治地方和普通行政地方
## 行政区划的概念、内容、特点
- 含义  
    特定国家按照一定原则和程序将其领土分成若干不同层次的区域，以便设置相应地方国家机关分层管理，实现国家职能的法律制度  
    概念核心是领土划分  
    与结构形式的关系：是国家结构形式的具体体现  
    作为一项制度包括的内容有：划分原则、划分主体、划分主体的权限、划分程序、划分结果、边界争议处理  
    - 特点：鲜明阶级性、历史范畴、受社会和自然条件制约、具有层次性、具有承载功能
    - 意义：涉及纵向国家权力的调整、配置，是国家宪法体制的重要组成部分，其规范、科学是国家民主政治发展的重要标志
## 我国行政区划
### 我国行政区划制度的内容
1. 行政区划原则：民族团结、有利于经济发展、便于人民参加管理、照顾历史
2. 行政区划主体
    - 全国人大：设立省级行政区和特别行政区
    - 国务院：省级行政区划界线确定、市级、县级行政区划设立  
    - 省级人民政府：乡、镇建制及区域划分
3. 行政区划变更程序  
    包括设立、撤销、更名、行政区行政中心驻地迁移  
    1. 省自治区直辖市变更由全国人大审议决定  
    2. 省自治区直辖市行政区域界线变更、自治州县自治县市直辖市的变更和隶属关系的变更、自治州县自治县市政府驻地的迁移、自治州自治县行政区域变更、县市行政区域界线重大变更、涉及海岸线海岛边疆要地重要资源地区及特殊情况地区的隶属关系或行政区域界线的变更  
    3. 县市市辖区的部分行政区域界线的变更，国务院授权省自治区直辖市政府审批，批准变更时报送民政部备案。乡民族乡镇的变更由省自治区直辖市政府审批
    4. 行政公署、街道办事处的变更由依法批准设立该派出机关的政府审批
4. 行政区划结果  
    >第三十条　中华人民共和国的行政区域划分如下：  
　　（一）全国分为省、自治区、直辖市；  
　　（二）省、自治区分为自治州、县、自治县、市；  
　　（三）县、自治县分为乡、民族乡、镇。  
　　直辖市和较大的市分为区、县。自治州分为县、自治县、市。  
　　自治区、自治州、自治县都是民族自治地方。

    >第三十一条　国家在必要时得设立特别行政区。在特别行政区内实行的制度按照具体情况由全国人民代表大会以法律规定
5. 行政区划争议解决  
    《行政区域边界争议处理办法》《行政区域边界争议处理条例》《关于区划管理的规定》
### 我国行政区划的复杂性
1. 名称多样：参考《宪法》第三十条
2. 层次多样：各种开发区
3. 名称与层次交叉：多种层级市（直辖市、地级市、县级市）；多种城市圈概念（长三角珠三角）
4. 与国家权力机关层次纠结
5. 与行政区划上国家机关领导级别高低（权力大小）纠结
### 计划单列市  
国家社会与经济发展计划单列市：大连青岛宁波厦门深圳  
1980s出现，享有省级经济管理权限，收支直接与中央挂钩，由中央财政与地方财政两分，无需上缴省级财政，行政区官员品级为副省级  
与直辖市的区别：直辖市是一级行政区，计划单列市只有经济上有省级待遇
### 民族自治地方  
#### 民族自治地方的含义
是在国家统一领导下，在少数民族聚居的地方，设立民族自治机关行使自治权，管理地方国家事务和本民族内部事务，以实现少数民族人民当家做主的一项制度  
核心是保障少数民族人民当家作主、实现各民族平等团结共同繁荣  
是民族自治与区域自治的结合
原因：历史传统与现实条件
#### 设立的种类
由一个、两个、三个或以上少数民族聚居为基础建立的自治地方  
我国目前有5个自治区30个自治州124个自治县  
命名：除特殊情况外，按地方名称+民族名称+行政地位
#### 与普通行政地方的异同
- 相同：都是中央统一领导下地方行政区域，国家机关采取相同组织及活动原则
- 不同：民族自治地方国家机关除享有一般地方国家机关职权外，还享有自治权，自治机关人员组成有特殊要求
#### 民族自治地方的自治机关及其自治权
- 自治机关：民族自治地方的人民代表大会和人民政府
- 自治权：制定自治条例和单行条例；经上级国家机关批准停止或变通执行国家法律和政策
- 自治机关人员组成：人大常委会主任或副主任应当由实行区域自治的民族公民担任，人民政府首长由实行区域自治的民族公民担任
### 自治条例和单行条例的含义
- 自治条例：由民族自治地方人民代表大会依《宪法》和《民族区域自治法》规定制定的关于本地方实行区域自治的组织和活动原则、自治机关的组成、职权以及自治地方有关重大问题的综合性规范文件  
- 单行条例：民族自治地方人民代表大会根据当地民族的政治经济文化特点，制定的关于某一方面具体事项的规范性文件
### 特别行政区
#### 特别行政区的含义
根据我国宪法和有关特别行政区基本法的规定，在我国领土范围内设立的，具有特殊法律地位，实行特别的政治、经济和法律制度的地方  
特别行政区是指香港和澳门  
特点：政治经济法律制度等方面与其他行政区不同，特殊性通过两个基本法作出全面规定
#### 特别行政区与中央的权力分配关系
- 特别行政区享有的权利  
    行政管理权、立法权、独立司法权和终审权、经中央授予的自行处理有关外交事务、可享受中央授予的其他权利
- 中央对其享有的权力  
    外交权、国防事务权、任命行政长官和主要官员（国务院）、决定特别行政区进入紧急状态、解释基本法（全国人大常委会）、修改基本法（全国人大）

# 国家标志
## 国家标志的意义
- 国家标志是国家的外在符号，对国家具有重要意义  
- 便于国际事务中国家之家的交往  
- 宣誓国家主权，维护国家尊严  
- 增强社会成员对国家的认同感
