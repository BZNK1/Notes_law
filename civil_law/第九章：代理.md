# 概念
>第一百六十一条  民事主体可以通过代理人实施民事法律行为。  
依照法律规定、当事人约定或者民事法律行为的性质，应当由本人亲自实施的民事法律行为，不得代理。

代理：一人代另外一人实施法律行为，其所产生的法律效果直接/间接归属于被代理的人  
代理的性质是代理人处于被代理人的位置实施行为，法律后果与被代理人自己实施时相同  
代理源于需求
- 私法自治之扩展  
    社会现实，经济生活之需求（时间精力能力有限）
- 基本形态：作出/接受意思表示（积极/消极代理通常情况下是结合的）  
    代理制度使得人格的自由发展成为可能，被代理人（本人）的影响力增加，没有代理制度，现代经济生活将无法进行
- 私法自治之补充  
    法律原因（缺乏行为能力）  
## 注意
- 事实行为、侵权行为不适用代理
- 准法律行为类推适用代理
- 某些法律行为不得代理（不得代理的法律行为不因追认而生效）
    结婚、离婚、收养、遗嘱、与人格权或者人格利益密切相关的
## 与其他制度的区分  
1. 使者（狗）  
    将已经确定的意思表示/已经做成的意思表示传达,或者接受意思表示  
    重要区别:无权决定意思表示的内容/意思形成的自由  
    使者就是工具，不是法律关系中的一方，传达意思表示无需行为能力要件  
    有疑问时，认定是否为使者取决于相对人是否认为其是使者（是否有决定自由）  
    在接受意思表示时，消极代理和传达难以区分，重点是看面貌，区分的意义在于意思表示对接收人/代理人生效时间不同  
    不得代理的行为可以使用使者
    ### 注意
    如果传达人表现的形象违背指示的扮演了代理人的角色，或者代理人违背指示的扮演了传达人的角色，这种扮演并不重要，只要行为符合内部指示的内容，法律行为就是有效的  
    在传达人超越传达权限扮演了代理人的角色，适用无权代理的规定  
    代理人无意识地扮演了传达人的角色时，适用传达错误规则
2. 法定代表人  
    法人实施法律行为可以通过法定代表人，也可以通过职务代理  
    代表：自然人以法定代表人身份出现时，不是独立的主体，而是法人的机关/代理：代理则是独立主体，不是法人的机关  
    代表人的行为就是法人的行为，代理人的行为并非法人的行为，只是效果由法人承受  
    职务代理可由单独委托或者公司章程产生  
    代表：越权代表行为不得对抗善意第三人/代理：越权代理的法律行为效力待定  
    代表：无需单独授权/代理：需要授权或者法律规定
3. 中介人  
    >第九百六十一条  中介合同是中介人向委托人报告订立合同的机会或者提供订立合同的媒介服务，委托人支付报酬的合同。

    中介人无代理委托人订立合同的权利，委托不一定会产生代理关系  
4. 行纪  
    >第九百五十一条  行纪合同是行纪人以自己的名义为委托人从事贸易活动，委托人支付报酬的合同。 

    >第九百二十五条  受托人以自己的名义，在委托人的授权范围内与第三人订立的合同，第三人在订立合同时知道受托人与委托人之间的代理关系的，该合同直接约束委托人和第三人；但是，有确切证据证明该合同只约束受托人和第三人的除外。  

    >第九百二十六条  受托人以自己的名义与第三人订立合同时，第三人不知道受托人与委托人之间的代理关系的，受托人因第三人的原因对委托人不履行义务，受托人应当向委托人披露第三人，委托人因此可以行使受托人对第三人的权利。但是，第三人与受托人订立合同时如果知道该委托人就不会订立合同的除外。  
    受托人因委托人的原因对第三人不履行义务，受托人应当向第三人披露委托人，第三人因此可以选择受托人或者委托人作为相对人主张其权利，但是第三人不得变更选定的相对人。  
    委托人行使受托人对第三人的权利的，第三人可以向委托人主张其对受托人的抗辩。第三人选定委托人作为其相对人的，委托人可以向第三人主张其对受托人的抗辩以及受托人对第三人的抗辩。

    以谁的名义：间接代理和直接代理问题
5. 冒名行为  
    不属于代理，不适用无权代理的规定  
    区分被冒用的名字对于相对人来说是否重要  
    - 不重要（姓名不具有区分意义，尤其是被冒名人的支付能力）：行为在冒名人和相对人之间生效  
    - 重要（姓名具有区分意义）：类推适用无权代理的规定  
        追人的：被冒名者和相对人间有效.拒绝追认的，冒名者对善意相对人承担无权代理的法律后果
# 代理法律关系 利益格局
被代理人（本人）通过无权代理制度得到保护  
通过基础关系/代理权授予代理人  
代理人以被代理人名义实施法律行为 
第三人（信赖保护问题，间接代理选择权，无权代理赔偿请求权，表见代理） 
# 代理的类型
| 区分标准 | 具体类型 |
|:----|:----|
| 代理权发生的原因 |委托代理、法定代理|
|显名与否|直接代理/间接代理|
|代理人的选任|本代理/复代理|
|代理权之有无|有权代理/无权代理|
|代理人行为内容|积极代理/消极代理|
|代理权行使的方式|单独代理/共同代理|
|是否被限定|概括代理/限定代理|
# 直接代理的要件
1. 代理的合法性（允许代理）  
    >第一百六十七条  代理人知道或者应当知道代理事项违法仍然实施代理行为，或者被代理人知道或者应当知道代理人的代理行为违法未作反对表示的，被代理人和代理人应当承担连带责任。

    >第一百七十七条  二人以上依法承担按份责任，能够确定责任大小的，各自承担相应的责任；难以确定责任大小的，平均承担责任。

    >第一百七十八条  二人以上依法承担连带责任的，权利人有权请求部分或者全部连带责任人承担责任。  
    连带责任人的责任份额根据各自责任大小确定；难以确定责任大小的，平均承担责任。实际承担责任超过自己责任份额的连带责任人，有权向其他连带责任人追偿。  
    连带责任，由法律规定或者当事人约定。
2. 公开性（显名代理）  
    为了保护相对人，代理人必须以被代理人的名义行为
    ## 特别情形
    - 表示受领人知道或者应当知道表意认为第三人行为（超市售货员）
    - 表意人只用说明以他人名义即满足公开性要求，无需具体说明是谁（债务未履行时，交易相对人要求表意人披露，表意人拒绝披露，类推适用无权代理规定）
    - 表意人因口误说成自己的名义，不适用意思表示错误规则
3. 代理人的行为（意思表示）
4. 代理权
    ## 代理权的产生
    1. 依据法律规定产生
        >第二十三条  无民事行为能力人、限制民事行为能力人的监护人是其法定代理人。
    2. 依据法律行为产生
        基础关系（委托合同、合伙合同、公司章程）——授予代理权
    3. 依据外表关系产生（依据法律产生）
        >第一百六十三条  代理包括委托代理和法定代理。  
        委托代理人按照被代理人的委托行使代理权。法定代理人依照法律的规定行使代理权。
        -  委托代理权产生
            委托代理权通过**需要受领**的**不要式**单方意思表示（授权行为）产生
            >第一百六十五条  委托代理授权采用书面形式的，授权委托书应当载明代理人的姓名或者名称、代理事项、权限和期限，并由被代理人签名或者盖章。
            - 授权行为具有**单方性**
            - 授权行为独立于**基础关系(邀约和承诺)（分离原则）**（如委托合同）
            - 授权行为的效力不受基础关系的影响 **（抽象原则）**（尤其是效力瑕疵不对代理权产生影响）  
            授权行为不能设定义务，基础关系才能为代理人设定义务
            >第一百六十四条  代理人不履行或者不完全履行职责，造成被代理人损害的，应当承担民事责任。  
            代理人和相对人恶意串通，损害被代理人合法权益的，代理人和相对人应当承担连带责任。
        >第一百七十二条  行为人没有代理权、超越代理权或者代理权终止后，仍然实施代理行为，相对人有理由相信行为人有代理权的，代理行为有效。
    ## 代理权的消灭
    >第一百七十五条  有下列情形之一的，法定代理终止：  
    （一）**被代理人**取得或者恢复**完全**民事行为能力；  
    （二）**代理人**丧失民事行为能力；  
    （三）代理人或者被代理人**死亡**；  
    （四）法律规定的其他情形。
